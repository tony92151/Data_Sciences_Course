## 備轉容量

在輸電網絡中，備轉容量或運轉儲備（operating reserve）是用來應付發電機故障、需求預測誤差和其他突發情況所預留的發電量。電力公司通常以最大發電機的發電量加上峰值負載的若干百分比作為運轉儲備的下限。
