## final project 

 [iMet Collection 2019 - FGVC6](https://www.kaggle.com/c/imet-2019-fgvc6)

iMet Collection 2019競賽是2019年CVPR的FGVC6研討會的一部分，屬於細粒度視覺識別挑戰賽，用於重點檢測自然物體及人造物體的細粒度視覺細節的圖像分類模型。數據由紐約大都會藝術博物館提供超過20萬件物品。
