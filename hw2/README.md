## One-hot encoding

如果是類別資料用1, 2, 3來表示第1類、第2類、第3類，會因為第3類比第1類數值還要大，在計算中會造成離原點較遠，因此使用one-hot encoding來處理。

> 第1類 表示成 100
> 第2類 表示成 010
> 第3類 表示成 001
